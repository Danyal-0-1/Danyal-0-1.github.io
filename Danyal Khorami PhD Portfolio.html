Mesquite MoCap

Open-Source Wearable Motion Capture System

üì∏
üì∏

Mesquite MoCap is a real-time, full-body motion capture system built with 15 wireless ESP32-C3 IMU nodes. Designed for accessibility and open-source transparency, it achieves 32 FPS capture rate with 15ms latency and 99.7% packet delivery. The system exports to BVH format and has been benchmarked against OptiTrack (¬±2.5¬∞ joint error).

Technical Highlights: Arduino/ESP32 firmware with time-synchronized networking, real-time sensor fusion algorithms, WebXR integration for live preview, optimized serial communication for embedded constraints.

To Wilt

MFA Thesis Installation: Emotion, LLMs, & Temporal Transformation

üåπ
üé®

"To Wilt" is a research-driven 4-space installation exploring how Large Language Models express, simulate, and question emotion through extended dialogue and sensor-fusion environments. The work challenges anthropomorphic AI assumptions while celebrating human temporal perception.

Spaces: (1) 8-screen video wall with 3D rose-growth animation + fine-tuned LLM philosophical dialogue, (2) Real-time LLM conversation printed live on networked dot-matrix printers, (3) 22k-image time-lapse decay dataset with LLM analysis, (4) Sensor-instrumented rose tank reflecting emotional decay through environmental metrics.

Happenstance

Image-to-Avatar Pipeline for Virtual Worlds

üë§
üë§

Happenstance is an end-to-end pipeline for converting single human photographs into rigged, animation-ready 3D characters for VR/XR environments. Uses Facebook Research's PiFUHD for high-resolution implicit function-based 3D reconstruction, followed by automated rigging in Mixamo.

Current Development: Improving mesh quality post-processing, automating Mixamo integration, reducing end-to-end latency for real-time applications, exploring diffusion-based clothing and accessory generation.
