Interactive experiment

Abstract motion capture lattice.

Rotate/zoom the structure to see how data streams synchronize; represents IMU packets and neural feature flow.

Drag to rotate · Scroll to zoom · Touch supported

Methodology

Nodes = IMU trackers + joints · edges = fusion relationships. Colors pulse based on simulated packet timing and confidence. Built with instanced meshes for performance.

  • Adaptive quality: device pixel ratio capped for mobile.
  • Fallback: degradable to 2D SVG if WebGL is unavailable.
  • Data hooks: drop in real BVH/CSV streams via viz.updateFromStream.
