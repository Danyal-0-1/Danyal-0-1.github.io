# Danyal Khorami — GitHub Pages (Static)

This is a no-build static site designed for GitHub Pages.

## Deploy (GitHub Pages)
1. Create a repo (e.g., `danyalkhorami.com` or `portfolio`)
2. Upload these files to the root of the repo.
3. In GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` / root
4. Your site will be available at the Pages URL.

## Custom domain (optional)
- Add a `CNAME` file in the repo root with:
  `danyalkhorami.com`
- In your domain DNS, point to GitHub Pages per GitHub instructions.

## Editing content
- Update `assets/data/profile.json` for the homepage/CV text.
- Update `assets/data/projects.json` for projects.
- Add images later and link them from each project (optional).
